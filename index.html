HTML

  • The head element contains information about the webpage also known as Metadata
  • The body element represents the visible content shown to the user
  • Metadata is not seen by users, but contains important information about accessibility, search engines, and performance.
  • Elements such as ul, ol, and il represent unordered lists, ordered lists, and list items.
  • The br element creates a line break in between two blocks of content

CSS

  • A margin indicates how much space we want around the outside of the element.
  • A padding indicates how much space we want around the content inside an element.
  • If you want to apply a rule to all the elements visible on the page, use a wild card symbol or *
  • We can combine two selectors into a single CSS rule with comma, allowing the declaration to affect both selectors.
  • We must link external CSS files to HTML files for it to show up.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add: adds modifications in current working branch
  • git -A flag indicates that we want to add all changes
  • git branch: shows all the local branches of your repo. The starred branch is the branch you are currently in.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • When using the if statement, code will only execute if the statement meets a condition, or is true.
  • In JavaScript and other programming languages, arrays are zero-indexed and sequential. Meaning, the first item in an array has an index of 0, not 1.
  • A for loop contains 3 important statements: the starting point for the loop, the condition, and the statement that allwows the array to iterate over each item.
  • A function is basically instructions that tells the computer how to perform a certain task.
  • A functions name is followed by a partentheses ().
  • For a function to show, it must be called by typing the name of the function including the parentheses.
